  <h1 align="center">InfluenceConnect - Projeto de Gerenciamento de Campanhas Publicitárias com Identificação de Influenciadores </h1>
</br>

<h2>Objetivos do Projeto</h2>
<li>O InfluenceConnect Frontend é responsável por proporcionar uma experiência de usuário intuitiva e eficiente, facilitando o gerenciamento de campanhas e a interação com influenciadores.</li>
<h2>Tecnologias Utilizadas</h2>
<img src="https://github.com/serratec/2023_2_Info4_InfluenceConnect_frontend/assets/141193350/031987f7-18ba-4d02-888c-141b39fcbf2c" height="40" width="40"> TypeScript;
<div><img src="https://miro.medium.com/v2/resize:fit:960/1*h7GMfvHKvLPkynG9NvULJw.gif" width="50" height="50">ReactJS;</div>

<h2>Configuração e Execução</h2>
<li>Clone o repositório..</li>
<li>Certifique-se de ter o Node.js e o npm instalados.</li>
<li>Instale as dependências utilizando npm install.</li>
<li>Execute o aplicativo React usando npm start.</li>

<h2>Tela de Login:</h2>
<li>Todos os tipos de usuários serão direcionados para a tela de login.</li>
<li>Administradores, empresas e influenciadores farão login com suas respectivas credenciais.</li>
